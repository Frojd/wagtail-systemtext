# Wagtail System Text

## Requirements

- Python 2.7 / Python 3.5+
- Django 1.8+
- Wagtail 1.7+


## Contributing

Want to contribute? Awesome. Just send a pull request.


## Installation

Install the library with pip:

```
$ pip install wagtailsystemtext
```
